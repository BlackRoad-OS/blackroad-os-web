## Brand Constraints

All layouts follow these visual guidelines:

- **Background**: Dark / black (#000, #0A0A0A, #111)
- **Accents**: Neon highlights
  - Primary: Orange/Yellow (#FF9D00, #FF6B00)
  - Secondary: Pink/Magenta (#FF0066, #FF006B, #D600AA)
  - Tertiary: Purple/Blue (#7700FF, #0066FF)
- **Typography**:
  - Headers: JetBrains Mono or similar monospace
  - Body: Inter or similar clean sans-serif
- **Spacing**: Generous whitespace, breathable layouts
- **Motion**: Subtle animations, no jarring transitions
